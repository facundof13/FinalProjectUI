About

What is this website?

This website is my final project for my CSC 4370 Web Development class.

How was this website made?

This website uses Angular and Bootstrap on the front-end. The server is written in Nodejs and uses a framework called ExpressJs. The database for this web application is MySQL.

How does it work?

I utilized a library for the front-end and the back-end called Socket.io. This library allows real-time updates to be sent from the client to the server and vice-versa, without the need to reload. This allows the chat and the player's moves to appear on all clients in real-time.

How do I play?

First create a room from the top of the page. Once in a room, copy the link and share with someone else. Have them paste the link into their browser and play tic-tac-toe.
